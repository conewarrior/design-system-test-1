# 금칙어 관리 페이지 기획서

> PDF p.4, p.20 기반
> ※ PDF에 "금칙어 키워드 관리 화면 추가 예정"으로만 되어 있음
> ※ 메뉴만 만들고 내용은 비워둠 (기획 미정)

---

## 개요

- **경로**: `/banned-keyword`
- **메뉴**: 사이드바 5번
- **목적**: 금칙어 키워드 등록/관리 (검색어 자동완성, 인기 검색어에서 제외)
- **상태**: 기획 미정 - 메뉴만 노출

---

## 메뉴 설명 (PDF p.4)

> 금칙어 키워드를 등록
> - 검색어 자동완성에서 해당 키워드 제외
> - 인기 검색어 노출에서 해당 키워드 제외
