## 개요

- **경로**: `/product`, `/product/:id`
- **메뉴**: 사이드바 3번
- **목적**: KT알파(기프티쇼) 전체 상품 조회 및 네이트온 서비스 노출여부 설정

---

## 페이지 목록

| 경로 | 페이지명 | 설명 |
|------|---------|------|
| `/product` | 상품 목록 | 전체 상품 목록 조회 |
| `/product/:id` | 상품 상세 | 상품 상세 정보 조회 (노출여부만 수정 가능) |

### 검색 필터

| 필드명 | 컴포넌트 | 옵션/플레이스홀더 | 검색 방식 |
|--------|----------|------------------|----------|
| 상태 | Select | 전체 / 정상 / 품절 / 판매중지 | exact |
| 노출여부 | Select | 전체 / 노출 / 미노출 | exact |
| 상품id | Input | "상품id" | exact |
| 상품명 | Input | "상품명" | like %검색어% |

### 액션 버튼

| 버튼명 | Variant | 동작 |
|--------|---------|------|
| 상품 동기화 | primary | 기프티쇼에서 신규 상품 가져오기 / 기존 상품 변경사항 업데이트 |
| 엑셀 다운로드 | secondary | 목록 엑셀 다운로드 |

### 테이블 정의

| 컬럼명 | 필드 | 타입 | 너비 | 정렬 |

### 상태 Badge 정의

| 상태 | Variant | 색상 |
|------|---------|------|
| 정상 | success | 초록 |
| 품절 | warning | 노랑 |
| 판매중지 | error | 빨강 |

### 테이블 동작

- **행 클릭**: `/product/:id`로 이동 (상품 상세)
- **체크박스**: 일괄 노출여부 변경 기능 (기능 필요여부에 따라 삭제 가능)
- **노출여부 토글**: 목록에서 바로 on/off 가능 (클릭 시 행 이동 방지)
- **정렬**: 최신 상품ID순 (내림차순)
- **페이지네이션**: 20개씩

### 수정 가능 항목

- **노출여부 토글만 수정 가능**
- 나머지 정보는 모두 조회만 가능

### mms 미리보기

- 해당 상품이 MMS에 발송되는 형태의 미리보기
- 필수 아니며, 개발팀 의견에 따라 제외 가능

## 데이터 모델

### 상품 (product)

```typescript
interface Product {
  id: number;                    // 상품 ID (네이트온)
  giftishowId: number;           // 기프티쇼 상품 ID
  brandName: string;             // 브랜드명
  productName: string;           // 상품명
  productImage: string;          // 상품 이미지 URL
  status: '정상' | '품절' | '판매중지';  // 상태
  productType: string;           // 상품타입 (쿠폰 등)
  detailType: string;            // 상세 상품타입 (커피 등)
  cancelable: boolean;           // 취소 가능 여부
  extendable: boolean;           // 연장 가능 여부
  refundable: boolean;           // 환불 가능 여부
  supplier: string;              // 공급사
  exchangeLocation: string;      // 상품 교환 장소
  searchKeywords: string;        // 검색키워드
  sellingPrice: number;          // 판매가
  originalPrice: number;         // 정가
  validityDays: number;          // 사용 유효기한 (일)
  saleEndDate: string | null;    // 판매 종료일
  isVisible: boolean;            // 노출여부 (네이트온에서 설정)
  createdAt: string;             // 등록일시
  updatedAt: string;             // 업데이트일시
}
```

## 비즈니스 규칙

1. **상품 동기화**
   - 기프티쇼의 상품목록을 배치로 가져옴
   - 수동 동기화 버튼으로 신규 상품 가져오기 / 기존 상품 변경사항 업데이트 가능

2. **노출여부**
   - 네이트온 서비스에서만 설정하는 값
   - 목록에서 토글로 바로 변경 가능
   - 상세 화면에서도 변경 가능

3. **상태값**
   - 기프티쇼 상품 상태값을 그대로 따름
   - 정상 / 품절 / 판매중지
